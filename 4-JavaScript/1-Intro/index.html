JavaScript - Intro

What is JavaScript?

  • Used for making the webpages alive/interactive
  • JavaScript codes usually embedded with HTML
  • No need for seperate compiler or compilation tool
  • All the browsers comes with in-built JavaScript engine, we can run and test the JavaScript in browser only. Ex: Chrome => V8, Firefox - SpiderMonkey
  • JavaScript is a interpreted programming language (compiles the code line by line, at the time of typing itself)
  • JavaScript's are most commonly called as "Scripts"
  • Earlier it was introduced to work with webpages (Front-end), later it is implemented in back-end (Node.js)
  • Java and JavaScript are different

What we can do with JavaScript?

  • Add new HTML to a page, change the existing content, modify styles
  • Add actions such as click, scroll on the HTML element in the form of "events"
  • Send a request and receive a response over the network
  • Also store some of the users info in the browser in the form of cookies, localStorage, sessionStorage etc...

What can't be do by JavaScript?

  • JavaScript can't able to access your file storage
  • Other than browser, JavaScript cant able to access anything in your device
